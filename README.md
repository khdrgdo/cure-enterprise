نظام متكامل لإدارة الصيدليات مبني بـ **Python + CustomTkinter**، يدير المخزون والمبيعات والتقارير والموردين والعملاء.

---

## 🚀 المميزات

### 🧾 نقطة البيع (POS)
- إضافة منتجات بسرعة عبر الباركود أو البحث
- عرض المنتجات في بطاقات مع الكمية والسعر (باستخدام `tk.Frame` خفيف الوزن للأداء)
- Pagination — 50 منتج / صفحة مع أزرار سابق / تالي
- سلة مشتريات تفاعلية مع تحكم بالكمية
- دعم الخصم وإجمالي الفاتورة
- طباعة الفاتورة على طابعة حرارية (ESC/POS) أو عرضها في نافذة PDF
- تصميم فاتورة احترافي: رأس ملون، جدول بألوان متناوبة، بطاقة معلومات الدفع

### 📦 المخزون
- إضافة / تعديل / حذف الأدوية
- دعم الباركود (بحث وطباعة باركود عبر ESC/POS)
- تصنيف الأدوية حسب الفئة
- تنبيهات عند انخفاض المخزون (Min Stock)
- سلة المهملات لاسترجاع المنتجات المحذوفة
- Pagination (50 منتج / صفحة)

### 🔄 المرتجعات
- استرجاع منتجات فردية من الفاتورة
- استرجاع الفاتورة كاملة
- بحث بالفواتير حسب الرقم أو التاريخ
- إرجاع الكميات تلقائياً للمخزون
- تمرير سلس (CTkScrollableFrame)

### 📊 التقارير
- 4 بطاقات أداء: إجمالي المبيعات، COGS، إجمالي الربح، صافي الربح
- رسم بياني أعمدة للمبيعات اليومية (tkinter Canvas) — مع زر إظهار/إخفاء
- جدول الأدوية الأكثر مبيعاً (Top 14)
- جدول توزيع طرق الدفع
- تصدير PDF (ReportLab)
- تصدير Excel (OpenPyXL)
- تقارير X (ملخص اليوم) و Z (إغلاق اليوم)

### 💰 الحسابات (Ledger)
- إدارة الإيرادات والمصروفات
- تصنيف تلقائي للمشتريات
- عرض الرصيد الجاري
- تقارير الأرباح والخسائر

### 👥 المستخدمين
- 3 أدوار: مدير النظام، صيدلاني، كاشير
- صلاحيات مخصصة لكل دور
- إضافة / تعديل / حذف المستخدمين
- إعادة كلمة المرور

### 📋 الآجل (Credit)
- فواتير آجلة مع حالة معلقة / معتمدة
- إدارة الدفعات
- تسديد دفعات جزئية

### ⚙️ الإعدادات
- 4 سمات: Dawn (داكن), Bloom (فاتح), Emerald (أخضر), Midnight
- تحجيم الخط
- رفع شعار الصيدلية
- تغيير اسم الصيدلية والعنوان
- عملة التقارير

## 🔧 التثبيت

### من الملف التنفيذي

1. افتح `install.ps1` كمسؤول (Run as Administrator)
2. أو شغل في PowerShell:
   ```powershell
   powershell -ExecutionPolicy Bypass -File install.ps1
   ```

### من السكريبت المصدري

```bash
pip install customtkinter tkcalendar pillow reportlab openpyxl
python Cure_Enterprise_Pro.py
```
